## MVVM
- Tech-stack
  - Jetpack
    - LiveData - подписка на изменение данных
    - ViewModel - хранение и управление данным, учитывая жизненные циклы
  - Flow - реализация реактивного потока
  - Retrofit - сеть
  - OkHttp - сеть
  - Picasso - загрузка изображений
  - Kodein - DI
  - Timber - логгер
  - Realm - база данных
